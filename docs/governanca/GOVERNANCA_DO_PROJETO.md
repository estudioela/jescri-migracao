# GOVERNANÇA DO PROJETO

## Objetivo

Este documento define as regras de governança do projeto Influencia.

Seu objetivo é garantir continuidade entre sessões, padronizar o trabalho entre diferentes IAs e manter a documentação organizada.

---

# Princípios

1. Fonte Única da Verdade (Single Source of Truth)

Uma informação deve existir em apenas um local.

2. Separação de Responsabilidades

Cada documento possui apenas uma responsabilidade.

3. Estado ≠ Histórico

O estado atual nunca é utilizado como histórico.

O histórico nunca representa o estado atual.

4. Sessões são descartáveis

Toda informação importante deve permanecer na documentação, nunca apenas na conversa.

5. Governança acima da ferramenta

A documentação deve funcionar independentemente da IA utilizada (Claude, GPT, Gemini etc.).
