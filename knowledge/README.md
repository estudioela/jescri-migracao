# knowledge/

Base de conhecimento de apoio ao desenvolvimento do **ELÃ | influência**
(`tear-v2-app/`, Laravel + React) — pesquisa de referência e baseline de
arquitetura/domínio que não são, por si só, a fonte oficial de estado do
projeto (essa é `docs/_workspace/TASK_ROUTER.md`, ver `CLAUDE.md`).

## Estrutura

- **`sistema-b/`** — baseline de arquitetura, modelo de domínio e fluxos de
  negócio do `tear-v2-app` (`ARCHITECTURE.md`, `DOMAIN_MODEL.md`,
  `BUSINESS_FLOWS.md`).
- **`references/`** — pesquisa técnica profunda usada na fundação do
  projeto (OAuth/Google Identity, sessão e autorização, modelo de
  identidade, onboarding/bootstrap).
- **`archive/`** — pesquisa e artefatos de processo já consumidos por
  decisões/documentos oficiais; mantidos por referência, não vigentes.
